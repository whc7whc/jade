🏪 JADE 賣家功能測試

注意：這些功能需要先登入並擁有賣家權限才能正常使用。

💰 銷售報表（賣家中心）

在賣家中心查看您的銷售數據、訂單統計和收入分析

功能特色：

  • 📊 銷售總覽（總收入、訂單數、平均訂單金額）
  • 📅 日期範圍篩選
  • 📈 報表類型選擇（每日/每週/每月）
  • 🔍 訂單狀態篩選
  • 📋 詳細訂單列表與分頁
  • 💾 CSV 匯出功能
  • 📱 響應式設計
  • 🎯 整合在賣家中心側邊欄中
進入賣家中心

進入後請點擊左側「銷售報表」功能

🎫 優惠券管理（賣家中心）

在賣家中心建立和管理您的店鋪優惠券

功能特色：

  • ➕ 新增優惠券
  • ✏️ 編輯現有優惠券
  • 🗑️ 刪除優惠券
  • 📊 使用統計
  • 🎨 折扣類型選擇（百分比/固定金額）
  • ⏰ 有效期限設定
  • 🎯 最低消費設定
  • 🏪 整合在賣家中心側邊欄中
進入賣家中心

進入後請點擊左側「優惠券管理」功能

或直接訪問：

獨立優惠券頁面

🔧 技術實現

API 整合方式：

  • 🔐 動態 seller ID 獲取（透過 ApplySeller API）
  • 🔄 多重 API 端點嘗試機制
  • 🛡️ 錯誤處理與 fallback 機制
  • 📱 響應式 UI 設計
  • 💾 本地儲存整合
  • 🎨 Bootstrap 5 樣式

📝 使用說明

  1. 確保已登入 JADE 帳號
  2. 確認擁有賣家權限（isSeller = true）
  3. 點擊上方按鈕進入相應功能頁面
  4. 系統會自動獲取您的賣家 ID
  5. 如果 API 無法連接，會顯示測試資料
